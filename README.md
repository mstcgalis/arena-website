2. In app.js replace `'xvburak-online'` with a channel slug of your choice. (Slug is the last part of channel link -> Channel: https://www.are.na/are-na-commons/animation-superchannel; Slug: `animation-superchannel` )
3. `npm install`
4. `npm start`
